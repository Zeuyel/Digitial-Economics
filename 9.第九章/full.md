# 数字经济统计

# 章前导语

数字经济统计体系的建立和完善有助于准确评估数字经济发展态势、衡量数字经济贡献、监测数字经济运行情况，对于各国推动数字经济健康可持续发展具有重要意义。然而，随着现代数字技术与人类经济社会发展、生产生活方式、生态文明建设等各领域、全过程的深入融合，数字经济这一新经济形态正在改变国民经济的生产、生活、消费和分配方式，传统国民经济核算体系在统计口径、核算方法等方面已无法满足实践需要。为此，如何立足现行数字经济统计工作实际，聚焦数字经济统计需求，构建全面、准确的数字经济统计体系已成为重要研究课题。

目前，国际上关于数字经济统计的内涵、思路、方法等方面尚未形成统一体系，诸多国际组织机构和国家尝试根据数字经济发展特征进行统计实践，为数字经济统计理论的完善提供了有益补充。中国数字经济统计发展起步较晚，但在数字经济产业分类、规模核算、指数构建等方面也进行了诸多实践，数字经济统计的“中国方案”正在不断探索和完善之中。

# 学习目标

理解数字经济统计的内涵与挑战；

理解数字经济统计的思路与方法；

了解数字经济规模核算和指数编制的国际经验；

了解中国数字经济的统计实践。

# 9.1 数字经济统计的内涵与面临的挑战

# 9.1.1 数字经济统计的范围

数字经济统计建立在明晰数字经济内涵的基础上，数字经济内涵将直接影响数字经济统计范围的界定。作为数字技术催生的新经济形态，数字经济的统计内涵和外延在由信息经济、互联网经济到数字经济的发展过程中不断拓展。目前，数字经济统计的范围尚未形成通用、统一的界定，但根据数字经济的概念和统计核算的可操作性，不同国家和相关国际组织所界定的数字经济统计的范围可大致划分为窄口径和宽口径。

# (1) 窄口径。

窄口径的数字经济统计围绕数字产业化展开，主要包括直接与数字信息技术相关的产业部门，如 ICT、电子商务、数字服务等，侧重统计依赖数字投入进行生产活动的产业部门。在统计实践中，国际货币基金组织（IMF）侧重统计窄口径下的“数字部门”，即数字化核心产业、ICT 产品和服务、数字平台以及平台支持的经济活动。① 美国经济分析局（Bureau of Economic Analysis, BEA）构建的经济测度框架体系②也属于窄口径的数字经济统计，统计范围主要包括与互联网和 ICT 产业相关的经济活动，统计分类包括数字基础设施、电子商务和数字媒体三类。具体而言，数字基础设施指支持计算机网络和数字经济的存在和运行所需的软件和硬件基础设施；电子商务指由数字平台和数字支付促成的商品交易或服务交易；数字媒体指人们在数字设备上创建、访问、存储或查看的内容。加拿大统计局（2019）、澳大利亚统计局（2019）以及英国数字、文化、媒体和体育部（2020）等对数字经济的统计也借鉴和沿用了窄口径。

# (2) 宽口径。

宽口径的数字经济统计主要围绕数字产业化和产业数字化展开，包括所有依赖数字投入或因使用数字投入而显著增强的经济活动，涵盖了国民经济各行各业的数字化发展以及利用这些数字投入的所有生产者和消费者。在统计实践中，以中国信息通信研究院（简称“中国信通院”）、中国国家统计局为代表的机构主要采用宽口径对数字经济进行统计核算。其中，中国信通院的数字经济统计方法被纳入G20《数字经济测算工具箱》①，其测算方法最初主要包括数字产业化部分和产业数字化部分；数字产业化部分包括电子信息制造业、基础电信业、互联网行业软件服务业等；产业数字化部分指传统产业应用数字技术带来的产出增加和效率提升，包括但不限于工业互联网、量化融合、智能制造、车联网、平台经济等融合性新产业、新模式、新业态。《中国数字经济发展白皮书》（2020）还进一步将“两化”拓展为“四化”框架，即数字产业化、产业数字化、数字化治理与数据价值化。

# 9.1.2 数字经济的层级结构

数字经济的定义既需要在概念上具有灵活性，也需要在统计上具有明确性。一方面，不同机构、组织或研究者需要灵活的定义来满足不同的需求和目标；另一方面，数字经济统计实践需要可供准确测量的明确要素，这些要素可以满足统计核算的可行性。为此，OECD（2020）根据数字经济的内涵，从统计角度明确了数字经济的层级结构，为不同统计方法构建了统一可比的框架，有助于深入认识和理解数字经济的统计内涵。数字经济的核算标准主要包括四个层级结构：核心层、狭义层、广义层以及数字社会。具体而言，核心层包括生产ICT产品和ICT及信息服务等的经济活动；狭义层包括核心层以及依赖数字投入所产生的经济活动；广义层包括核心层和狭义层以及因使用数字投入而显著增强的经济活动；数字社会包括数字经济的前三个层级以及未被统计在GDP内的其他数字化交易和活动，如免费数字平台服务等（见图9-1）。

# 9.1.3 数字经济统计面临的挑战

目前数字经济统计面临的首要挑战在于缺乏对数字经济概念的统一定义，因而难以对数字经济进行国际比较。在统计实践中，传统统计体系主要以国民账户体系（System of National Accounts, SNA）为基础，以GDP核算为中心，形成了涵盖经济、金融、财政、贸易、投资等多个领域的一系列统计标准、手册和规范。随着数字技术以新理念、新业态、新模式等全面融入人类经济、政治、文化、社会、生态文明建设各领域和全过程，传统统计体系也面临巨大挑战。

# (1) 数据作为关键生产要素难以核算。

随着数字经济时代数据规模的爆发式扩大，数据已成为驱动经济社会发展的新型关键生产要素，非结构化的数据也重构了统计理论的本源。首先，不同于货物或服务的特征，数据具有无形性、共享性和生产消费统一性的特征，数据产权模糊、资产范围扩大导致数据资产的定义、范围、权属等难以界定，这给生产、投资、消费和收入等领域的统计工作均带来了挑战。其次，传统统计数据的获取对象主要为结构化数据，但随着数字经济时代的发展，非结构化数据逐渐成为主流，数据资源海量化和高频化、数据来源和结构复杂化、数据产权分离化等导致传统统计手段和制度的适应性不足，传统统计体系采集和处理非结构化数据的效率低下。最后，传统国民经济核算尚未将数据价值列入生产资本进行统计，但数据带来的价值可能远高于利用永续盘存法等传统方法所得出的估值。数据的共享性赋予了免费数字商品隐性价值。尽管消费者没有直接支付费用，但消费者和生产者可以通过隐性交易来实现数据的资产价值，然而如何衡量这一价值仍是数字经济统计面临的难题。

# (2) 数字经济生产与消费的边界难以界定。

传统国民账户体系明确界定了生产者和消费者的身份以及生产和消费的范围，但随着数字经济为经济活动提供越来越多的便利，身份以及生产和消费的边界也变得更加模糊。

一方面，随着数字技术和数字平台的普及应用，生产的门槛降低了，消费者很容易就能成为产品供应者和价值创造者，导致生产和消费的边界模糊。例如，滴滴、饿了么、美团等数字平台能为消费者或灵活就业人员提供中介服务，使非企业化的单位和个人得以通过数字平台服务参与生产活动，改变了生产者和消费者之间的互动方式。这种生产和消费边界的模糊化使商品和服务的价格及交易量测算变得困难，从而增加了数字产出和交易的测算难度。另一方面，数字化使生产主体和生产关系逐渐呈现虚拟化和网络化趋势，部分住户部门可以将自给性产品提供给市场，这也导致了商品的消费和投资属性的边界模糊化。例如，消费者可以利用闲暇时间在滴滴、爱彼迎、闲鱼等数字平台获取收入。但传统国民账户体系并未将自给性服务包含在统计范围内，这给数字经济生产与消费的统计核算带来了挑战。

# (3) 新业态新模式的统计标准存在盲点。

数字经济的发展催生了以数字技术为基础的新业态新模式，颠覆了传统产业的市场模式、组织形态和管理模式，对国民经济行业分类、统计产品分类等标准提出了巨大挑战。首先，免费经济的出现让消费者能免费或以低廉的价格获得数字商品或服务，也让企业可以通过提供免费商品的方式来吸引流量，进而通过广告等其他方式盈利。但目前的GDP核算未能体现这种免费商品的生产价值和消费价值，这对生产和消费的统计标准提出了挑战。其次，数字技术极大地降低了信息搜寻成本，减少了信息不对称，推动了共享经济不断发展壮大，但共享经济涉及的产业部门、机构单位以及核心项目还不明确，新业态下总产出和增加值、最终消费和资本形成、进口和出口等条目的统计标准有待进一步梳理和规范。最后，数字化转型推动了实体经济与数字经济的深度融合，数字平台的存在导致一些经济活动和产品在统计中难以准确归类，对行业和产品分类标准提出了挑战。

# 9.2 数字经济统计的思路与方法

# 9.2.1 数字经济统计的思路

根据数字经济的内涵，数字经济统计的重点包括数字经济规模、数字经济要素投入、数字经济新业态新模式等方面。

# (1) 数字经济规模的统计思路。

目前数字经济规模的统计思路主要包括两种：一是国民经济核算框架下的GDP核算法。这种方法的统计思路主要是在界定数字经济内涵和范围的基础上，基于现行的国民经济核算框架，采用投入产出法、支出法等统计方法，对不同行业数字部门的增加值或总产出进行加总，从而统计出数字经济的规模。但由于统计数据可得性的限制，部分数据需通过假设法、计量经济模型法、增长核算法等方法进行推算。二是综合评价框架下的指数编制法。这种方法的统计思路主要是基于数字经济的不同维度，构建系统、全面的多层次评价体系指标，涵盖新媒介、新服务、新交易形式等数字经济活动，进而构建反映数字经济发展的综合评价体系。由于不同评价体系的数字经济分类和指标选取可能存在不同，因此不同综合评价框架下的数字经济测度结果也有所差异。

# (2) 数字经济要素投入的统计思路。

目前数字经济要素投入的统计内容主要包括数字资本投入、数字劳动投入以及数据资产。其统计思路大致可以归纳为：首先界定不同数字经济要素投入的范围和分类，其次构建数字经济要素统计框架，最后结合现有统计方法核算相应的数字经济要素投入总量。具体而言：第一，数字资本投入主要反映数字经济活动中投入资本的价值变化情况，通常将资本类型划分为建筑物、机器设备、其他资本、ICT 硬件和软件等。统计数字经济要素时，需将数字资本从传统资本中剥离出去，然后采用永续盘存法等方法进行测算，目前这仍是数字经济要素投入统计的难点。第二，数字劳动投入主要包括工作内容涉及数字经济商品或服务的劳动者。统计数字经济要素时，需明确数字经济劳动者的性别、年龄、薪资、学历等劳动特征，然后进行统计测算。第三，数据资产指由企业拥有或者控制的、能够为企业带来未来经济利益的、以物理或电子方式记录的数据资源。由于其性质复杂、核算难度高、工作量大，目前数据资产的统计核算仍处于探索阶段。

# (3）数字经济新业态新模式的统计思路。

数字经济的快速发展带动了免费经济、共享经济等大量新业态新模式的快速发展，本部分主要以免费经济为代表介绍数字经济新业态新模式的统计思路。免费经济的统计思路主要包括基于实物转移的统计思路和基于易货交易的统计思路（见图9-3）。基于实物转移的统计思路为：首先对免费商品的生产者虚拟一笔产出（免费商品），该产出以实物转移的方式免费提供给消费者；其次对消费者虚拟一笔实物转移收入来消费生产者提供的免费商品，再对消费者虚拟一笔消费支出。基于易货交易的统计思路为：首先对生产者虚拟一笔产出来提供免费服务，对消费者虚拟一笔产出来提供顾客价值，双方通过数字平台进行易货交易；然后对生产者虚拟一笔收入以获得消费者价值，对消费者虚拟一笔消费以获得免费服务。

# 9.2.2 数字经济统计的方法

现行数字经济统计体系常用的统计方法主要包括三种：GDP 核算法、指数编制法以及卫星账户法。

# (1) GDP 核算法。

GDP 核算法首先基于数字经济的内涵和范围确定数字经济产业统计分类，然后利用经济统计年鉴、经济普查数据、投入产出表等数据，核算数字经济核心产业各行业的增加值，最后加总得到数字经济核心产业的增加值。GDP 核算法常用于核算数字经济核心产业的增加值，主要包括生产法和支出法。生产法以总产出与中间投入之差测算数字经济各行业增加值，通过加总数字经济各行业增加值衡量数字经济规模；支出法先定义数字经济支出领域，然后加总各领域支出，最终测算出数字经济规模。在统计实践中，生产法运用最广，美国、加拿大、澳大利亚等国家主要采用该种方法进行数字经济统计。

由于数字经济各行业的详细数据不全，因此部分行业增加值、行业中数字经济增加值等需进一步估算，如BEA在假定数字经济中间消耗占数字经济总产出的比重与相应产业中间消耗占总产出的比重相同的前提下，对数字经济核心产业的增加值进行估算。

# (2) 指数编制法。

指数编制法常用于对数字经济发展情况进行综合评价，可操作性强、应用范围广，国际电信联盟、OECD、欧盟、世界银行、中国信通院、上海社会科学院等国内外组织和机构采用不同的方法和指标构建了不同的数字经济指数。指数编制法主要通过构建多级指标体系，利用统计加权等方法对多种指标进行合成，进而得到综合指数。例如，OECD构建了数字经济统计指标体系，包括智能基础设施投资、社会推进、创新能力、ICT促进经济增长与增加就业岗位4个一级指标，以及38个二级子指标。欧盟在此基础上进一步优化，构建了数字经济与社会指数（DESI），包括宽带接入、人力资本、互联网应用、数字技术应用、数字化公共服务程度共5个一级指标，12个二级指标，以及31个三级指标。

# (3) 卫星账户法。

卫星账户是传统国民经济核算的辅助统计账户，通常用于统计国民经济中特殊形态的经济模式。数字经济卫星账户将数字经济活动运行视为整体，试图厘清数字经济活动中不同主体运行发展的特征，确保数字经济统计的全面性和整体性。目前数字经济卫星账户法是数字经济统计领域的前沿方法，还没有国家系统地编制完整的数字经济卫星账户体系，但OECD、BEA、新西兰统计局等组织的工作为数字经济卫星账户的编制奠定了基础。OECD从数字交易的性质出发构建了数字经济卫星账户的基本框架（见图9-4），该框架按交易性质是否为数字订购或数字交付对数字经济进行刻画，大多数国家和学者在这一框架下对数字经济卫星账户进行了拓展和细化。

组织机构开始尝试编制不同数字经济指数来度量数字经济发展状况，反映数字经济发展趋势。

# 9.4.2 数字普惠金融指数

近年来，依托人工智能、区块链、云计算、大数据等数字技术，新型数字金融已经成为普惠金融的重要原动力和增长点。数字普惠金融通常泛指传统金融机构与互联网公司利用数字技术从事融资、支付、投资和其他新型金融业务的模式。相比于传统金融，数字普惠金融提供的金融服务更直接、覆盖面更广，大幅提高了金融服务的可得性、便利性和普惠性。为了科学、准确地刻画我国数字普惠金融的发展现状，北京大学数字金融研究中心、上海新金融研究院和蚂蚁金服集团联合课题组利用蚂蚁金服的海量数据，编制了覆盖中国31个省（自治区、直辖市）、337个地级以上城市（地区、自治州、盟等）和约2800个县域（包括县级市、旗等）的北京大学数字普惠金融指数。这一指数主要从互联网金融服务的覆盖广度、使用深度和数字支持服务程度三个维度来构建指标体系（见表9-7），为国内外各界的数字普惠金融指标体系实践提供了重要参考。

根据北京大学数字普惠金融指数（2011—2020年）报告的研究结果，主要可以得出以下结论：第一，中国数字普惠金融经历了高速发展到高质量发展的转变，尽管近几年发展速度有所放缓，但2011—2020年指数值平均每年增长了  $29.1\%$  ，其主要驱动力由数字普惠金融的数字化程度增长逐渐转变为数字金融使用深度增长。第二，数字普惠金融发展的第一梯队主要包括上海、北京和浙江，而第三梯队的省份主要地处西部地区或东北地区。第三，不同地区间的收敛速度差异明显，但近两年差异有所缩小，主要原因是使用深度的地区差异收敛系数有所反弹。

# 本章小结

基于数字经济的内涵和统计核算的可操作性，数字经济的统计范围可大致划分为窄口径和宽口径。其中窄口径主要围绕数字产业化展开，宽口径主要围绕数字产业化和产业数字化展开。数字经济的层级结构可进一步划分为核心层、狭义层、广义层以及数字社会。数字经济面临的统计挑战主要包括：数据作为关键生产要素难以核算；数字经济生产与消费的边界难以确定；新业态新模式的统计标准存在盲点。

数字经济统计的内容主要包括数字经济规模统计、数字经济要素投入统计、数字经济新业态新模式统计等。现行数字经济统计体系常用的统计方法主要包括GDP核算法、指数编制法以及卫星账户法。

在国际实践中，各国主要通过数字经济规模核算和数字经济指数编制评估数字经济发展状况。数字经济规模核算包括以美国等欧美国家为代表的窄口径核算框架，以及以中国为代表的宽口径核算框架。数字经济指数编制基本采用多层级指标构建体系，但不同数字经济指标体系所反映出来的同一国家的数字经济发展水平存在较大差异。
