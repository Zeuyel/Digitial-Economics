# 数字经济监管

# 章前导语

数字经济的发展正在重构经济社会的运行方式，它在促进消费者福利改善、企业效率提升和社会有效治理的同时，也带来了诸多新的风险，典型的如网络与数据安全风险、个人隐私泄露风险、市场垄断风险等，因此需要政府介入，对数字经济进行监管。数字经济监管是确保数字经济发展全面提高社会总福利的根本途径。数字经济监管是多元主体为实现公共利益目标而采取多种手段的系统治理活动。数字经济监管最主要的特点是：它涉及多元主体，具有多种治理机制，并不局限于政府通过行政手段实施的监管。由于政府监管是数字经济监管的核心，因此政府监管体制根本上决定了数字经济监管的定位、手段及其有效性。特别是在中国行政主导的体制下，政府行政监管始终是核心。本章的分析围绕政府监管体制来展开。由于数字经济具有独特的发展规律和市场失灵形成机制，存在更为多元的经济社会损害风险，因此数字经济监管面临更为复杂的环境和诸多新的问题，为此，需要深入分析数字经济发展中各种市场失灵的形成机制，并有针对性地制定科学的监管政策。

掌握网络与数据安全监管；

掌握个人隐私数据保护；

# 15.1 网络与数据安全监管

在数字经济时代，网络是国家主权的新空间，数据主权是国家主权的新体现。维护网络与数据安全关乎国家主权和公共安全。有效的网络与数据安全监管能为数字经济发展提供重要的安全保障，实现发展与安全的协调。

# 15.1.1 网络安全与数据安全

网络安全是指网络系统中的硬件、软件及数据受到保护，不因偶然的或恶意的原因而遭到破坏、更改、泄露，能够连续、可靠、正常地运行，确保网络服务不中断。网络安全包括网络设备安全、网络软件安全和网络信息安全。确保网络安全的重点是防范各种网络攻击，常见的网络攻击有口令窃取、欺骗攻击、缺陷和后门攻击、认证失效、协议缺陷、信息泄露、指数攻击（病毒和蠕虫）和拒绝服务攻击等。因此，网络安全的核心是强化技术保障，有效应对各种网络攻击，确保网络系统安全、可靠和有弹性地持续运行。

数据安全是指在数据信息的整个生命周期中保护数据信息免遭未经授权的访问，并抵御损坏、失窃、泄露等风险，确保数据的可用性、完整性和保密性。数据安全包括从硬件和存储设备的物理安全到管理和访问权限控制，直至软件应用程序的逻辑安全等方面面，以及确保数据安全的组织政策、管理程序等安全制度。我国《数据安全法》第三条指出：“数据安全，是指通过采取必要措施，确保数据处于有效保护和合法利用的状态，以及具备保障持续安全状态的能力。”数据安全监管的目的是确保数据处于有效保护和合法利用的状态，防止各种未经授权的非法数据接入、数据盗取和数据破坏。

经济学研究网络与数据安全监管时，重点是采用经济学方法来分析网络与数据安全风险，并设计激励性网络与数据安全监管政策。因此，从经济学的意义上来说，网络与数据安全监管追求以有效率的方式来实现网络与数据安全的目标，即网络与数据安全监管不应以牺牲数字经济创新发展为代价，而应更好地促进数字经济创新发展。因此，网络与数据安全监管应坚持以安全促发展、以发展保安全的基本原则。

# 15.1.2 网络与数据安全监管的需求

# （1）网络与数据安全监管是维护国家主权和发展利益的需要。

第一，网络与数据安全监管是维护国家公共安全与社会稳定的需要。网络与数据安全是数字经济发展的基石，是维护国家公共安全的重要领域。随着数字经济的发展，经济社会活动日益网络化，网络空间成为新的国家战略空间，数据资源日益成为世界各国抢占的战略要地，数据安全成为影响数字经济发展和国家安全的重大问题。因此，迫切需要加快构建网络与数据安全监管体系，提升网络与数据安全监管能力，维护国家公共安全。

第二，网络与数据安全监管是维护国家经济安全的需要。网络与数据安全是国家经济安全的重要基础。2020年，全球因网络犯罪造成的直接货币损失约为1万亿美元，而在网络安全方面的支出超过1450亿美元，上述两项成本共占当年全球GDP的  $1.3\%$  。全球网络犯罪造成的总经济成本（直接经济成本、间接经济成本、上游系统成本）在2020年约

为4万亿美元，约占全球GDP的  $4\%$  。①随着中国经济体量的日益扩大，网络与数据安全风险日益增大，这就要求国家对网络与数据安全进行监管。

第三，网络与数据安全监管是促进数字经济高质量发展的需要。网络与数据安全是数字经济高质量发展的重要基础。随着数字经济的高速发展，经济活动日益网络化和数字化，维护网络和数据安全就是维护国家经济安全，从而为数字经济高质量发展提供重要的基础保障。科学的网络与数据安全监管将极大地促进数字经济高质量发展。网络与数据安全监管可以更好地保护企业的数据资产和商业秘密，从而鼓励企业从事数据驱动的创新业务；有助于维护消费者隐私数据安全，增强消费者的消费信心，营造可信任的消费环境，促进数字商业的发展；能保障数据共享流通的安全、可控，有助于促进数据要素的市场化配置，为数字经济发展提供可靠的市场基础。

# (2）网络与数据安全供给存在市场失灵。

第一，网络与数据安全具有公共品属性。在数字经济中，网络成为经济社会运行的重要基础设施，生产、消费、社会运行等都建立在网络的基础上。维护网络与数据安全具有广泛的经济社会影响。由于网络数据安全是多个主体共同努力的结果，因此个别主体会有“搭便车”的激励。根据经济学理论，维护网络与数据安全具有广泛的社会价值，但是其需要付出非常高的成本，且这一成本主要是由数字商务企业来承担，因此企业缺乏投资的激励。

第二，网络与数据安全风险具有负外部性。网络与数据安全风险往往是系统性风险，一旦发生重大网络安全风险事故，将会给众多利益相关者和整个社会带来严重的负外部性。同时，一个企业投资于网络与数据安全也会对他人和社会产生正外部性，给众多利益相关者带来益处。在网络与数据安全风险具有外部性的情况下，维护网络与数据安全的私人成本和社会成本不一致导致微观主体无法提供社会最优的网络与数据安全。

第三，网络与数据安全不同主体间存在信息不对称。网络与数据安全风险的发生往往具有较大的不确定性，各种网络犯罪往往是国际化的、远程的和隐秘进行的，网络运营者与数据持有者往往缺乏充分的信息，“猫和老鼠的游戏”成为网络与数据安全问题的常态。而且，为了商业利益或组织声誉，网络运营者与数据持有者往往并不会及时向利益相关者、监管机构或社会发布安全风险预警或公布安全事故信息，从而导致监管者和社会公众往往并不知晓潜在的安全风险。

第四，网络与数据安全问题本质上是激励扭曲问题。在市场失灵的情况下，由于成本和收益不对称，数字企业往往缺乏提供社会最优的网络与数据安全的激励，从而产生网络与数据安全保护不足的风险。因此，网络与数据安全问题不仅仅是一个网络安全技术保障问题，也是一个相关主体的激励机制设计问题。网络安全问题产生的根本原因往往是不完善的政策设计所产生的激励机制扭曲。有效的激励机制是网络与数据安全监管的核心。网络与数据安全监管，既需要强化技术标准、组织规则、技术保障要求等强制性监管手段，也需要通过划分主体责任、制定奖惩措施等监管机制设计来实行激励性监管。

# 15.1.3 中国网络与数据安全监管政策

党的十八大以来，以习近平同志为核心的党中央高度重视网络安全工作，不断加强国家网络安全战略谋划和顶层设计，推动中国网络安全体系不断完善、网络安全保障能力持续提升、网络安全屏障日益巩固。

# (1) 确立了以国家总体安全观为统领的监管理念。

总体国家安全观是网络与数据安全监管的总指引。总体国家安全观是习近平总书记于2014年4月15日在中央国家安全委员会第一次全体会议上提出的，它是新时代国家安全工作的总体指导思想。我国《网络安全法》第一条指出：“为了保障网络安全，维护网络空间主权和国家安全、社会公共利益，保护公民、法人和其他组织的合法权益，促进经济社会信息化健康发展，制定本法。”我国《数据安全法》第四条明确指出：“维护数据安全，应当坚持总体国家安全观，建立健全数据安全治理体系，提高数据安全保障能力。”

# (2) 构建了网络与数据安全监管的基础性法律体系。

目前中国基本构建起网络与数据安全监管法规体系，形成了具有中国特色的网络与数据安全法律体系。国家颁布了《网络安全法》《数据安全法》《个人信息保护法》等基础性、综合性法律。《网络安全法》确立了网络安全支持与促进措施，规定了保障网络运行安全、网络信息安全的要求，为维护网络和信息安全制定了规则；《数据安全法》建立了数据分类分级保护制度，集中统一、高效权威的数据安全风险评估、报告、信息共享、监测预警机制，数据安全应急处置机制、数据安全审查制度等数据安全制度，明确了数据处理者的数据安全保护义务，并为政务数据安全与开放提供了规则和方向；《个人信息保护法》则进一步细化、完善了个人信息保护原则，包括明确个人信息处理规则、个人信息跨境提供的规则、个人在个人信息处理活动中的权利、个人信息处理者的义务，并确立了以国家网信部门为核心的个人信息保护机构体制。此外，相关部门和地方政府还出台了一系列行政法规、部门规章、规范性文件、地方性法规、地方政府规章、国家标准等政策法规。

# (3) 不断健全网络与数据安全监管体制。

要通过构建完善的数据安全制度及数据跨境流动管理制度，不断探索和健全网络与数据安全监管体制，具体内容如下：一是建立数据分类分级保护制度。根据数据在经济社会发展中的重要程度，以及一旦遭到篡改、破坏、泄露或非法获取、非法利用，对国家安全、公共利益或者个人、组织合法权益造成的危害程度，对数据实行分类分级保护。关系国家安全、国民经济命脉、重要民生、重大公共利益等的数据属于国家核心数据，应实行更加严格的管理制度。二是建立数据安全风险监测机制。国家建立集中统一、高效权威的数据安全风险评估、报告、信息共享、监测预警机制，对数据安全实行有效的全过程风险管理。三是建立数据安全应急处置机制。发生数据安全事件，有关主管部门应当依法启动应急预案，采取相应的应急处置措施，防止危害扩大，消除安全隐患，并及时向社会发布与公众有关的警示信息。四是建立数据安全审查制度。基于安全审查制度，对影响或者可能影响国家安全的数据处理活动进行国家安全审查。五是建立数据跨境流动监管制度。加强对数据跨境流动的管理，重点加强对关键数据信息跨境流动的监管，以维护国家数据主权。同时，积极参与数据跨境流动全球治理体系的构建，推进跨境数据流动国际合作。

# (4) 重视网络与数据安全技术保障能力建设。

党的十八大以来，习近平总书记在关于网络安全工作的“四个坚持”中强调了大力发展网络安全技术保障能力的重要性，明确指出“要坚持网络安全教育、技术、产业融合发展，形成人才培养、技术创新、产业发展的良好生态”。党的十八大以来，中国网络与数据安全技术保障工作取得了初步成效，网络与数据安全技术保障能力建设主要从以下角度展开：一是加强网络安全技术保障法律法规体系建设。修订了《网络安全审查办法》，并发布了《信息安全技术关键信息基础设施安全保护要求》等30项网络安全国家标准。二是培养服务于网络与数据安全技术领域的专业人才。国家通过在有关高校开设网络安全相关专业来强化人才建设，基于人才建设提升网络与数据安全技术保障能力。三是推动网络与数据安全技术研发。国家致力于推动网络安全技术研发与产业化发展，促进监测预警、数据保护、可信体系构建等领域的基础理论创新和关键技术研发，推动大数据及人工智能安全技术、工业互联网安全技术等新兴领域研究，并通过加大产业投入、优化产业生态等手段加快与数据安全相关的高新技术的产业化，为网络与数据安全提供坚实的技术保障。

# 15.2 个人隐私数据保护

在数字经济中，数据是数字商务企业最重要的生产要素，大数据成为企业价值创造的最重要资产。数字经济服务提供商大量采集消费者用户的个人数据信息并对其进行加工和开发，以获取商业利益。但是，数字商务企业大量收集大数据的行为也产生了侵犯消费者个人隐私和个人隐私信息泄露的风险。因此，个人隐私数据保护成为数字经济监管的重要新课题。

# 15.2.1 隐私

隐私是个人重要的人格权，是公民个人的基本权益。厄尔·沃伦（Euri Warren）和路易斯·布兰代斯（Louis Brandeis）在1890年发表的论文《隐私权》中指出，隐私权是个人免受打扰的权利。①面对信息通信技术发展对个人隐私的影响，阿兰·威斯汀（Alan Westin）在《隐私与自由》一书中指出，隐私权是“个人自己决定何时、如何以及在何种

程度上向他人传递有关个人信息的权利”②，即控制个人信息的权利。因此，隐私保护也称个人信息保护。

在数字经济时代，数字技术使企业、政府等私人和公共组织得以大量采集用户个人数据并进行存储、加工和使用。由于这些数据直接或间接地反映了个人隐私信息，因此个人隐私主要体现为隐私数据。隐私数据被界定为“数据中直接或间接蕴含的，涉及个人或组织的，不宜公开的，需要在数据收集、数据存储、数据查询和分析、数据发布等过程中加以保护的信息”。我国《个人信息保护法》

# 15.2.2 隐私经济学

在数字经济中，个人数据是重要的经济资源，也是数字商务企业从事商业活动的重要投入要素，个人数据的基本功能是为市场运行提供信息。因此，隐私经济学属于信息经济学的一个分支，它将数据看作经济活动所必需的要素，其会对市场有效运行产生重要影响。消费者向商家披露个人数据信息并不总是带来坏处，它也会促使商家向消费者提供更个性化的服务，降低消费者的搜寻成本，提高消费者的消费体验，并降低市场运行成本。但是，企业过度采集和利用消费者个人数据会侵犯消费者隐私，给消费者带来财产损失和无形的心理伤害。

# (1) 芝加哥学派的自由市场观念。

芝加哥学派认为，个人隐私数据市场是完美的，个人隐私保护问题应该通过市场机制来解决，政府不应该干预。在市场有效的情况下，理性的个体之间通过私人之间的隐私数据交易谈判可以实现最优的个人隐私披露，从而实现个人隐私保护和个人数据开发利用的最佳组合。据此，芝加哥学派经济学家都坚持认为，市场信息越多，则资源配置效率越高，市场机制越能实现理想的隐私保护和市场信息的最优配置。由于理性的消费者和企业之间的私人谈判或市场交易可以实现最佳的个人数据信息披露，因此政府干预会降低市场效率。

芝加哥学派反对政府干预隐私问题，认为隐私保护监管会扭曲市场化的信息机制，产生三种效率损害效应：一是阻碍资源优化配置。信息自由流动是打造竞争市场和提高经济效率最重要的机制，而隐私保护监管会限制信息自由流动，提高市场信息不完全程度，增加市场运行的信息成本，阻碍市场实现有效的资源配置。二是阻碍有效率的交易的实现。隐私保护增强了信息不对称，会增加交易成本，阻碍有效率的交易的实现。三是产生分配不公平。隐私保护会削弱生产性投资激励，并带来分配不公平。

# (2) 个人隐私数据市场失灵理论。

目前，大多数经济学家都认为，个人隐私数据市场存在失灵，因此需要政府干预。原因主要包括以下几个方面：一是消费者的隐私决策并不是一种理性的行为。很多行为经济学的研究发现，消费者的隐私决策往往是非理性的。二是个人隐私数据市场存在信息不对称。消费者和商家之间在个人隐私数据信息上存在严重的信息不对称，即消费者通常并不知道商家如何使用自己的数据信息，因此无法实现最优的私人之间的隐私数据交易谈判解。三是私人之间的隐私数据交易谈判中各方的谈判力不对等。在私人之间的隐私数据交易谈判中，消费者和企业之间的谈判力明显不对等，这阻止了平等主体间谈判的市场均衡解的出现。四是个人隐私数据市场中存在数字平台垄断及策略性滥用行为。大规模采集和商业化消费者个人数据信息的数字平台往往具有非常强的市场支配地位，在市场竞争严重缺乏的情况下，支配平台未必具有向消费者提供较强隐私保护的激励。因此，支配平台并不必然具有促进隐私保护的激励，需要通过有效的竞争来激励支配平台保护消费者隐私。综上，个人隐私数据市场失灵理论强调政府通过实行有效的个人隐私数据保护来维护消费者权益，并促进个人隐私数据的安全采集和利用。

（3）隐私经济学研究的核心问题。

羽茜、隐私经济学研究的核心问题主要有以下几个：

一是消费者个人隐私支付意愿的识别。科学识别消费者个人隐私支付意愿是隐私经济学研究的重要基础性工作。制定和实施科学的个人隐私数据保护政策成为重点，而其基本前提是精准识别消费者的隐私支付意愿，即数字商务企业为了让一个消费者同意开放其个人隐私数据而支付的消费者愿意接受的货币补偿额度。由于消费者的个人隐私价值偏好受到一个国家的文化观念、特定的场景等因素的影响，并且具有较大的个体差异性和动态变化性，因此测算个人隐私支付意愿具有复杂性。对该问题，一方面需要引入行为经济学的分析方法来解释，另一方面需要进行大量的基于场景的消费者个人隐私价值偏好的经验分析。

二是大型数字平台侵犯隐私的风险及治理政策。由于大型数字平台是数字经济的重要主体，且大型数字平台都采用数据驱动的商业模式，大量采集和利用消费者个人数据，并实行个人化推荐、个人化定价等行为，因此，大型数字平台采集利用个人隐私数据会产生哪些经济社会影响，以及如何强化大型数字平台保护消费者隐私的责任成为数字经济学研究的核心问题。个人隐私数据保护会增加数字平台的合规成本，对数字平台数据驱动的商业模式和创新产生重要的影响。因此，要全面评估个人隐私数据保护政策的经济影响。

三是个人隐私数据保护成本和收益的权衡。隐私经济学主要关注如何平衡个人隐私数据保护与数据要素开发利用的关系，即合理确定个人数据披露和数据保护的边界，实现个人利益与公共利益的平衡。因此，个人隐私数据保护不是绝对禁止对个人数据的采集利用，而是在充分尊重个人隐私权的基础上合法依规采集利用数据。对于数字商务企业来说，过度采集消费者个人数据会带来基于数据开发的高收益，但是也会引起消费者对其的信任度下降，导致收益下降，所以在做决策时要权衡成本和收益。对于消费者来说，过度

# 15.2.3 个人隐私数据保护需求

目前，个人隐私数据保护主要是依据两个具有互补性的理论基础：一是强调个人隐私数据是个人人权的重要组成部分，是个人人格尊严的重要体现，因此需要政府介入。二是突出个人隐私数据的经济要素功能，认为个人隐私数据是数字经济活动的重要投入品，可以通过私人主体之间的谈判和交易来实现最佳的披露，但是由于市场机制存在失灵，因此需要政府介入以促进个人隐私数据的安全、有序采集和利用。

# (1) 个人隐私数据是重要的人格权。

在数字经济背景下，个人隐私数据是公民个人重要的人格权，是个人人格尊严的重要体现，因此应当受到保护。我国《个人信息保护法》第二条明确指出：“自然人的个人信息受法律保护，任何组织、个人不得侵害自然人的个人信息权益。”个人隐私数据侵害主要表现为数据商务企业或政府行政机关在未获得个人同意的情况下，过度采集和使用个人

隐私数据信息，以及因技术限制或安全漏洞而引发的个人隐私数据泄露，从而使当事人受到各种损害的行为。因此，隐私权强调公民个人对他人获取和使用个人数据的控制能力。各国都特别强调严格加强对个人敏感数据的保护，以充分保护个人的人格权。

我国《个人信息保护法》第二十八条和二十九条规定，敏感个人信息是一旦泄露或者非法使用，就容易导致自然人的人格尊严受到侵害或者人身、财产安全受到危害的个人信息，包括生物识别、宗教信仰、特定身份、医疗健康、金融账户、行踪轨迹等信息，以及不满十四周岁未成年人的个人信息。只有在具有特定的目的和充分的必要性，并采取严格的保护措施的情形下，个人信息处理者方可处理敏感个人信息。处理敏感个人信息应当取得个人的单独同意；法律、行政法规规定处理敏感个人信息应当取得书面同意的，从其规定。

(2) 个人隐私数据保护是数字经济发展的重要基础制度。

个人隐私数据是国家数据资源重要的组成部分之一，个人隐私数据安全是影响和制约数据要素价值释放的重要因素。对于消费者来说，在在线交易的过程中，如果数字商务企业大量实施未经消费者同意的数据采集和利用行为，并由此引起个人隐私数据泄露及相关的个人权益受到损害，则会引发消费者对在线交易的担忧，从而降低消费者对在线交易的信心，并最终影响数字经济的发展。科学合理的个人隐私数据保护会促进数据要素的依法依规采集和利用，有利于激励消费者合理地披露个人隐私数据，激励企业合法规范地使用数据，构建安全、可信任的在线交易环境，促进在线交易的健康发展。因此，个人隐私数据保护能够更好地促进数据要素的采集和利用，充分释放数据要素的经济价值，促进数字经济发展。

(3) 市场机制无法提供有效的个人隐私数据保护。

隐私经济学更多地将个人隐私数据看作一种经济品，认为它是一种生产要素。芝加哥学派认为，隐私数据是信息，信息是市场机制有效运行的重要条件。由于个人隐私数据披露既有成本也有收益，因此理性的个人和企业之间基于市场的个人隐私数据交易谈判可以实现最优的个人隐私数据保护和采集利用，此时政府的个人隐私数据保护政策往往会扭曲市场的有效运行，并降低社会总福利。但是，现实是：由于个人隐私数据市场失灵，市场机制并不能够实现最优的消费者个人隐私数据保护，因而需要政府适度介入，实施个人隐私数据保护政策。

第一，个人隐私数据市场存在严重的信息不对称。信息不对称主要表现为平台和消费者之间的数据信息采集和处理能力存在巨大差异，即数字平台通过多种方式采集和开发利用消费者个人数据，具有强大的数据采集和开发利用能力。但是，消费者对于数字平台在什么时候、以何种方式采集个人数据，个人数据将被运用到哪些领域和以何种方式被共享再用，以及个人数据被采集和利用会带来哪些影响等并不知情。在信息严重不对称的情况下，数字平台往往过度采集和利用消费者个人隐私数据。

第二，消费者非理性隐私决策。消费者非理性隐私决策主要是指“隐私悖论”，即大部分消费者都宣称非常在乎个人隐私数据，但在多数情况下其实际行为却与所宣称的不一致，即其往往并不会采取积极行动来保护自己的个人隐私数据。例如，为了获得折扣、促销优惠，消费者主动让渡个人隐私数据，或者忽视或根本不阅读隐私政策条款等。隐私悖论本质上是一种行为偏差，其导致消费者在隐私决策中无法做出最优的理性决策。同时，

消费者在披露个人数据时存在“隐私短视”现象，即对个人隐私数据的经济价值缺乏远见，更关注眼前利益，不充分评估个人数据披露带来的长期风险。

第三，消费者隐私数据披露存在负外部性。由于个人的隐私数据之间具有关联性，一个消费者向数字商务企业披露数据后，该企业可以依据这一数据来预测同类消费者的消费行为，从而产生个人隐私数据披露的负外部性。在存在负外部性的情况下，由于其他消费者的数据披露导致单个消费者的数据价值降低，因此单个消费者付出成本来保护个人隐私数据的收益极低，这会导致特定单个消费者缺乏保护个人隐私数据的激励，从而过多披露个人隐私数据并引起个人愿意接受的个人隐私数据补偿价格下降，进而导致企业过度采集数据。在存在负外部性的情况下，即使引入市场竞争也不会改变个人隐私数据被过度采集和利用的问题，甚至可能会导致消费者的状况恶化。

第四，个人隐私数据市场存在隐私数据滥用行为。隐私数据滥用是指数字商务企业收集消费者的个人隐私数据后利用算法等信息技术对数据进行挖掘，实施严重损害消费者福利的个人化定价、误导消费者做出购买决策等滥用行为，造成对消费者个人利益的损害。由于数字经济市场存在典型的赢家通吃或赢家多吃的现象，因此少数几个大型数字平台占据了非常大的市场份额，消费者如果不接受其提出的隐私数据条款，则无法使用相应的App。由此，消费者失去了与大型数字平台进行谈判的能力，无法有效抗衡大型数字平台过度采集和利用其数据的行为。

# 15.2.4 中国的个人隐私数据保护政策

面对数字经济时代日益增加的个人隐私侵犯风险，目前各国（地区）都在加强对个人隐私数据的保护：欧盟在2018年颁布了《通用数据保护条例》（General Data Protection Regulation, GDPR）；美国加州2020年制定了《加州消费者隐私法》（California Consumer Privacy Act, CCPA），该法成为美国个人隐私保护立法的重要模板；中国2021年8月20日第十三届全国人民代表大会常务委员会第三十次会议通过了《中华人民共和国个人信息保护法》，确立了中国个人隐私数据保护的基本制度。

# (1) 完善个人隐私数据保护立法。

我国《个人信息保护法》是统一个人信息利用规范、明确相关主体责任及法律救济原则的专门法，其内容体现了以下几个要点：

第一，确立个人数据处理的基本原则。我国《个人信息保护法》主要确立了以下重要原则：一是遵循合法、正当、必要和诚信原则；二是采取对个人权益影响最小的方式，限于实现处理目的的最小范围原则；三是处理个人信息应当遵循公开、透明原则；四是处理个人信息应当保证个人信息的质量原则；五是采取必要措施保障所处理的个人信息的安全原则；六是“知情一同意”原则。采集处理个人信息应当取得个人同意，个人应在充分知情的前提下自愿、明确做出决定。

第二，明确数据主体的权利。明确规定数据主体对个人信息的处理享有知情权、决定权、查看权、可携带权、更正权、删除权、撤回同意权等基本权利。

第三，明确个人信息跨境流动的规则。明确规定个人信息处理者向境外提供个人信息需通过国家网信部门组织的安全评估、经专业机构进行个人信息保护认证、按照国家网信部门制定的标准合同与境外接收方订立合同，同时确保境外接收方处理个人信息的活动符

合中国个人信息保护标准。关键信息基础设施运营者和处理个人信息数量达到国家网信部门规定数量的个人信息处理者，还须通过国家网信部门安全评估。

第四，建立隐私侵害救济机制。针对侵害个人信息的违法行为，我国制定了公益诉讼与行政规制两种救济途径，其中针对情节严重的违法行为，可责令整改，没收违法所得，并处五千万元以下或者上一年度营业额百分之五以下的罚款。

# (2) 加强个人隐私数据保护执法。

近年来，我国不断加强对个人信息保护的监管执法力度，主要通过约谈、专项治理等手段针对网络空间内的个人信息侵犯事件进行整治。例如，2018年国家网信部门就“支付宝年度账单事件”暴露的支付宝、蚂蚁信用违规采集和利用个人信息的行为约谈支付宝，要求其进行专项整顿。2020年，国家网信部门就圆通速递员工泄露消费者个人信息事件约谈圆通速递，并责令其建立运单数据管理系统。2022年，国家网信部门就网约车巨头滴滴非法采集和利用个人数据的行为开出了80.26亿元人民币的罚单。

# 15.3 数字市场反垄断

反垄断政策是数字经济市场有序运行的重要政策保障，科学的反垄断政策有助于维护自由竞争的市场体制，充分发挥市场竞争机制在促进资源优化配置和激励企业创新中的决定性作用，从而提高消费者福利与社会总福利，促进国家经济高质量发展，实现持久繁荣。因此，反垄断政策是市场经济国家的基础性经济政策。从全球各国反垄断政策的规定来看，反垄断政策主要是禁止三类垄断行为：价格合谋、损害竞争的并购和滥用支配地位的行为。

# 15.3.1 数字经济反垄断政策的基本导向

# (1) 坚守竞争政策的基础性地位。

市场竞争是促进数字经济高质量发展的重要动力，数字经济反垄断应坚守竞争政策的基础性地位，从根本上维护竞争性市场体制。尽管数字经济具有相对独特的经济规律，但竞争始终是促进经济高质量发展和消费者福利改善的基础机制，维护市场竞争机制和竞争过程始终是竞争政策的根本任务。一方面，竞争政策应积极有为。对于那些竞争损害证据明确的支配企业的滥用行为和严重损害消费者福利的不正当竞争行为，要及时采取有效的竞争政策加以纠正。另一方面，应坚守竞争政策的基础性地位。应重点消除严重扭曲市场竞争的产业政策、阻碍创新发展的政府监管政策和不作为的竞争政策。应确保竞争政策的基础性地位，营造公平竞争的市场环境，促进数字企业的创新发展。

# (2) 坚持消费者福利标准并重视对非价格因素的保护。

在数字经济背景下，仍应坚持反垄断和提高消费者福利标准。长期以来，反垄断法都以提高消费者福利和防止消费者福利受到损害作为根本目标，但基于芝加哥学派的反垄断政策都是将价格作为判断是否存在垄断行为以及是否损害消费者福利的主要依据。在数字经济中，消费者福利损害不再仅仅局限于价格方面。特别是在平台向消费者免费提供服务或数字产品的情况下，企业之间的竞争已经超越了单一的价格竞争模式。在平台免费供应

产品的情况下，消费者在选择时更加关注产品质量、产品多样性、隐私保护等非价格因素。数字经济反垄断仍应坚持消费者福利标准，但要对以价格为中心的消费者福利标准进行扩展，就要求反垄断政策从传统的以价格为中心转向重视对非价格因素，特别是质量、创新和隐私的保护。

# (3) 将促进创新作为反垄断的优先目标。

创新是维护数字经济市场竞争和促进数字经济高质量发展的最重要的驱动力量。在数字经济背景下，创新带来的动态效率提升对消费者福利和社会总福利改善具有特别重要的意义。在数字经济中，创新既是维护市场竞争的动力，也是市场竞争的结果。因此，维持企业创新的激励应该成为反垄断审查的最优先目标。创新优先的反垄断执法主要体现在以下几个方面：一是对于有利于创新的企业之间的合作行为不宜适用本身违法原则。在数字经济中，企业之间的数据共享协议、联合制定技术标准、数字版权合作组织等合作行为往往是促进创新的，应实行宽松的反垄断政策。二是为维持企业创新的激励，应谨慎干预收取高额许可费、拒绝许可和拒绝数据开放等行为。只有在案件所涉及的技术或数据满足该技术或数据是其他企业参与竞争所不可或缺的条件，并且其他企业无法合理复制的情况下，反垄断执法才应介入，并谨慎采取要求企业向竞争对手开放数据、履行强制的互操作义务、强制许可专利技术等救济措施。三是实施审慎的反垄断监管，强化市场研究和程序正义，防止过度监管和错误执法对竞争造成损害，特别是防止运动式执法和“一刀切”政策对企业创新发展造成的损害。数字经济反垄断监管不能以牺牲创新为代价，而应更好地促进数字经济创新。

# (4) 重点关注数据和算法等新型滥用行为。

数据垄断成为数字经济反垄断需要特别关注的问题。对数字平台数据驱动下的滥用行为的反垄断审查主要包括三个方面：一是非法的数据采集和利用行为。数字平台往往向消费者提供免费服务，但消费者同时也向平台提供了具有经济价值的数据。如果平台没有获得消费者授权同意就擅自采集消费者的个人隐私数据，或者迫使消费者接受不合理的个人数据采集使用条款，则构成一种剥削性滥用行为，这种行为是与传统垄断企业向消费者征收不合理的高价格相似的垄断行为。二是数据和算法滥用行为。某些数字平台实施的数据驱动的排他行为会影响市场竞争、阻碍创新，并影响用户隐私保护。三是数据驱动的企业并购行为。支配平台企业通过并购来获取和控制不同来源的数据可能会增强在位者的市场势力，弱化市场竞争，形成对竞争对手的封锁。

# 15.3.2 数字经济反垄断重点关注的垄断行为

各国反垄断法禁止的非法垄断行为主要包括价格合谋、滥用支配地位和企业并购等行为。在数字经济背景下，反垄断法重点关注一些新兴的垄断行为，主要包括算法合谋、算法个人化定价、“二选一”、拒绝接入数据，以及数字平台并购等行为。

# (1) 算法合谋。

在数字经济中，数字商务企业普遍采用大数据算法决策会产生价格合谋。算法定价明显提高了价格合谋出现的可能性。针对算法合谋的反垄断政策包括：

第一，合理界定反垄断意义上的非法协议。反垄断法应对合谋的“协议”采取更宽泛的界定，不是仅仅局限于企业之间存在明确的相互沟通的证据，而是更多关注“一致行

为”的事实和达成限制竞争的合谋结果的事实，并加强附加证据的获取，从而确保竞争规则的科学适用。

第二，明确当事企业的法律主体责任。对于算法合谋，反垄断的法律责任主体首先应该是当事企业，在合谋事实成立的情况下，当事企业不能以自己不知情或未主动参与来免责。因此，不管对于何种形式的算法合谋，反垄断责任主体都应该是定价算法的使用者，即当事企业。

第三，实行有效的算法事前监管。一是事前监管将算法设计放在突出的位置，坚持“基于设计来遵从法律”的原则，要求算法程序设计遵守和体现竞争原则，从源头消除算法合谋违法行为。基于算法设计来遵从法律的规则要求算法开发者和算法使用者通过算法设计来确保“竞争中立”，即算法设计应该确保其不会产生反竞争的合谋效应。二是探索建设人工智能算法认证制度和安全审计制度。政府应委托专业机构对人工智能系统进行安全认证，监管机构可以授权专业机构进行算法安全审查，同时要求系统使用者建立内部安全审查制度。

# (2) 算法个人化定价。

算法个人化定价也被称为“大数据杀熟”，是指企业基于大数据算法来对消费者实行个人化定价的做法，其会严重侵害消费者利益。针对“大数据杀熟”的反垄断监管政策的重点是：

第一，依据多部法律法规进行综合治理。算法个人化定价不仅涉及消费者权益和交易公平性问题，还涉及消费者隐私保护等问题。因此，需要依据《消费者权益保护法》《反不正当竞争法》《电子商务法》《个人信息保护法》等相关法律来执法。此外，政府还可以采取反垄断执法、隐私保护监管等行政行为，同时更多地采用市场化或技术性措施。

第二，采用合理推定执法原则，谨慎适用本身违法原则。经济学分析表明，算法个人化定价作为一种价格歧视方式，其并不必然损害消费者福利和社会总福利，更多的情况是一部分消费者利益受损，一部分消费者获益，对总消费者福利和社会总福利的影响是不确定的。因此，反垄断执法应坚持个案分析。

第三，突出透明度要求。算法个人化定价对福利的损害在很大程度上取决于消费者的行为反应，特别是消费者的信息掌握程度或信息获取难度。增强消费者的知情权和信息获取能力能显著化解支配平台算法价格歧视的竞争损害。为此，商家在使用消费者个人信息数据实行个人化定价时，应明确告知消费者，针对该消费者的个人化定价必须获得消费者的同意，并且消费者拥有退出权。

# (3) “二选一”。

在反垄断法中，“二选一”行为也被称为“独占交易协议”，即支配平台规定其合作商家只能与其进行交易或合作，不能与其竞争对手交易或合作。在数字经济中，支配平台实施的“二选一”行为具有独特的竞争损害效应，成为反垄断法的关注重点，国家市场监督管理总局重点查处的阿里垄断案、美团垄断案、知网垄断案都涉及“二选一”行为。

第一，封锁竞争对手。支配平台通过实施“二选一”行为实现对商家的锁定，并且通过交叉网络效应增强消费者对该平台的黏性，使竞争对手无法获得有效与其竞争的用户规模，从而达到封锁竞争对手的目的。

第二，强化对商家的剥削。“二选一”行为使平台商家失去了自由选择权和与平台讨

价还价的能力，增强了平台剥削商家的能力。

第三，促进寡头合谋。寡头平台实施“二选一”行为会产生“市场分割”效应，形成一种缓解价格竞争的寡头合谋机制，最终损害消费者的利益。

# (4)拒绝接入数据。

数据是数字企业获得竞争优势的重要战略资产。为了维持市场垄断地位，支配平台有激励实施拒绝竞争对手接入其拥有的数据的行为。根据数据占有企业与数据接入需求方之间的关系，支配平台实施的拒绝接入数据的行为主要有两种类型：横向拒绝接入数据和纵向拒绝接入数据。横向拒绝接入数据主要是相互竞争的企业之间拒绝接入数据，纵向拒绝接入数据主要是数据占有企业拒绝上下游互补性企业接入数据。

第一，支配平台实施的横向拒绝接入数据行为通常并不构成非法。根据反垄断法，在竞争市场中，单个企业没有义务向竞争对手提供接入数据服务的义务。一个企业拒绝竞争对手的数据接入请求并不会对市场竞争构成损害，不应受到反垄断法的禁止。反垄断法禁止的是具有“必要设施”属性的拒绝接入数据行为。美国和欧盟的竞争执法机构都主张，在大多数情况下，反垄断法不应强制要求企业与其竞争对手分享数据，只有在支配平台的数据构成必要设施，并且拒绝其他企业接入数据行为严重损害市场竞争时，反垄断执法机构才会对拒绝接入数据行为加以禁止。在数字经济中，由于数据具有明显不同于传统自然垄断行业“必要设施”的经济特征，在很多情况下，数据往往并不满足“具有不可复制性”这一要求，因此“必要设施”原理仅适用于少数例外情况。

第二，反垄断禁止的主要是支配平台实施的纵向拒绝接入数据行为。纵向拒绝接入数据是指支配平台拒绝向平台生态系统中的第三方合作商开放数据，此时平台与第三方合作商之间是互补关系，典型的如菜鸟网络拒绝顺丰接入数据。在支配平台是“守门人”，即支配平台是第三方商家与消费者实现交易或互动的必要通道的情况下，如果其纵向拒绝接入数据，那么第三方商家将无法与消费者接触，导致其被排除在市场之外。因此，纵向拒绝接入数据行为是一种典型的滥用市场支配地位的行为，应该受到反垄断法的禁止。

# (5) 数字平台并购。

并购是数字平台快速成长并实现跨界经营的重要方式，但是数字平台并购往往有较大的概率损害竞争。因此，目前各国都在重新制定数字平台的并购政策，对支配平台的并购实行更为严格的并购控制。数字平台并购带来的突出问题是会引起市场集中度的显著上升，尤其是那些旨在消除潜在竞争者的“杀手并购”，以及非横向的跨界并购。

第一，数字平台并购会削弱市场竞争。由于数字经济市场内在的高度集中化趋势，支配平台的并购往往会成为加剧市场集中和企业实现跨界经营的重要方式，从而严重影响正常的市场竞争。

第二，支配平台对初创企业的“杀手并购”会严重损害市场的潜在竞争。大型数字平台对构成潜在竞争威胁的初创企业的并购实际是一种消除潜在竞争的滥用行为，该行为不仅会损害市场的竞争性，也会严重阻碍数字市场的创新。

第三，数字平台跨界并购会造成生态垄断。例如，谷歌、亚马逊、腾讯、阿里等大型数字平台通过并购进入多个与其核心业务相关的业务领域或新的不相关的业务领域，产生了生态垄断势力。

# 15.3.3 数字经济反垄断新变化

# (1) 强化反垄断事前监管。

为了应对大型数字平台的市场势力逐渐变大和滥用垄断行为的问题，避免出现“执法不足”的风险，欧盟和美国反垄断执法部门在强化事后执法的同时，分别制定了具有针对性的《数字市场法案》和《美国创新与选择在线法案》来强化对大型数字平台的反垄断事前监管。

欧盟和美国的反垄断事前监管主要针对大型数字平台。欧盟的《数字市场法案》对平台“守门人”地位的认定标准为：一是具有强大的经济地位，对市场具有重要的影响；二是具有强大的中介地位，运营一个核心的服务平台并且该服务平台是商业用户实现与终端用户的交易的重要通道；三是具有稳固且可持续的市场地位。《美国创新与选择在线法案》对平台“守门人”地位的认定标准主要涉及平台规模、市场影响和中介地位。

欧盟和美国的反垄断事前监管的重点是明确“守门人”平台的行为规则。欧盟《数字市场法案》第五条和第六条对“守门人”平台提出了9项本身违法禁止行为（黑名单）和10项可抗辩本身违法行为（灰名单），《美国创新与选择在线法案》对“守门人”平台提出了3项本身违法禁止行为和7项可抗辩本身违法行为（见表15-1）。总体来说，欧盟和美国事前监管严格禁止的“守门人”平台行主要包括限制交易、强制使用平台服务、歧视性平台条款、最惠国条款、强制性服务捆绑等，基本适用本身违法原则；对于捆绑搭售、限制平台商家接入数据或互操作性、阻碍消费者用户数据携带、歧视性数据接入等行为则适用可抗辩本身违法原则，在平台企业有可信的证据显示该行为具有显著的效率效应的情况下，其不构成违法。

# 15.4 网络非法有害内容监管

在数字经济背景下，网络成为人们获取信息、交流信息和进行文化娱乐消费的主要渠道。网络信息内容的消费具有便捷性，其供给具有低成本、生产者与消费者合一等特点。因此，网络成为生产和传播淫秽色情信息、暴恐内容、谣言传言、极端思想、带有偏见的观点、侵犯版权的内容等非法有害内容的温床。由于互联网内容传播的速度快、范围大，大量非法有害内容的出现会产生较大的社会负外部性，造成严重的社会危害。因此，强化网络信息内容监管，营造清朗的网络空间成为数字经济网络监管的重要内容。

# 15.4.1 非法有害内容的界定

由于各国政治、文化和价值观存在差异，目前国际上并没有关于非法有害内容的统一界定，各国都根据本国的实际情况来界定网络信息内容监管的具体内容。欧盟委员会2018年出台了抵制非法网络内容的措施，其中指出，非法有害内容包括涉及恐怖主义、儿童性

侵、非法的仇恨演讲、侵犯版权或其他知识产权（如假冒商品、设计侵权）、非法商业行为（如诈骗、欺诈、订阅陷阱）、违反其他社会准则或服务条款等的内容。

英国政府 2019 年发布的《网络危害》白皮书根据有害内容的传播程度以及对个人和社会的影响，将有害内容分为三大类：第一类是“明确规定的有害内容”，包括涉及儿童剥削和性虐待、恐怖主义内容和活动、有组织的移民犯罪、现代奴隶制度、极端色情、复仇色情、骚扰或网络跟踪、仇恨犯罪、鼓励和协助自杀、煽动暴力、销售非法商品或服务、向 18 岁以下的人士发送不雅照等的内容；第二类是“含义不太清晰的有害内容”，包括涉及网络欺凌、极端主义、造谣、恐吓、暴力和鼓吹自残等的内容；第三类是与未成年人有关的有害内容，如色情、交友内容等。

中国将网络非法有害内容划分为非法内容和有害内容两大类。根据 2019 年 12 月国家互联网信息办公室发布的《网络信息内容生态治理规定》, 其第六条规定了 11 种禁止的非法内容 (违法信息), 主要包括危害国家安全、宣扬恐怖主义、宣扬邪教、煽动民族仇恨、散布色情或暴力信息等的内容; 第七条规定了 9 种有害内容 (不良信息), 主要涉及使用虚假标题、炒作绯闻、带有性暗示、煽动人群歧视、宣扬低俗等的内容。

总体来说，我们可以将网络信息内容监管的对象分为非法内容和有害内容。非法内容是指包含大量违反国家法律的内容，如影响国家政治安全、恐怖主义、盗版侵权、假冒伪劣商品、虚假广告、侵犯个人隐私、恐怖主义、色情暴力、儿童性侵等具有严重危害的违法内容。有害内容是指虽然没有严重违法但是会对其他人或社会产生一定损害的内容，包括网络霸凌、虚假新闻或信息、鼓励自残或自杀的内容，以及歧视性或侮辱性言论、网络骚扰、有违道德伦理的低俗内容等。

# 15.4.2 网络非法有害内容的生产机制

在数字经济中，网络信息内容的传播主要是通过大型数字内容平台来实现的，典型的如美国的 YouTube，中国的抖音、微博、小红书等。数字内容平台的盈利方式是典型的广告费模式，即平台通过向消费者提供免费内容来获得用户注意力，从而获得较多的用户流量，然后通过向广告商收费来获取盈利。因此，数字内容产业本质上属于“注意力经济”，数字平台经营的重点是通过内容供给来赢得用户注意力。

数字内容平台之间的竞争实际上是用户注意力的竞争，一个平台越能吸引用户，其用户基础就越大，用户点击量就越高，此时在该平台投放广告的效果就越显著，广告商就越愿意在该平台投放广告，由此平台通过在线广告获取的利润回报也就越高。由于很多非法有害内容迎合了消费者的独特偏好，能吸引受众的眼球，有利于吸引用户注意力和增加平台流量，因此数字内容平台往往对非法有害内容采取纵容的态度。

数字内容消费的网络效应使非法有害内容可以以更快速度在更大范围内传播。数字内容消费与其他产品消费的一个非常大的不同是存在明显的信息瀑布效应，即数字内容消费者往往更关心更多人关注或讨论的话题，即一个话题越流行，讨论、阅读的人越多，就会吸引越多人的关注。在数字内容消费具有网络效应和信息瀑布效应的情况下，数字内容平台之间的商业竞争会促使企业采取更加极端的差别化战略来吸引受众的注意力。由于数字内容平台采集了大量的用户数据，因此平台借助大数据分析技术可以精准识别每个消费者的偏好，然后通过大数据算法来向用户提供个性化的内容推送，即向特定消费者用户提供

其本人偏好的数字内容。这一方面降低了消费者的搜寻成本，另一方面也导致消费者用户接触到的信息内容受到了限制，从而产生“回音室”效应和“信息茧房”问题。“回音室”效应是指消费者更偏好与其价值判断或观点相近的内容，“信息茧房”则是指数字内容平台为了迎合消费者的偏好而只向其推送符合其偏好的内容，这两种效应的结合会导致消费者的认识更加狭隘或极端，对社会形成偏见。

# 15.4.3 网络非法有害内容本质上是一种负外部性问题

网络非法有害内容本质上属于负外部性问题。负外部性是指某个经济主体的经济行动给其他经济主体或社会带来损失，而该经济主体却不为此付出代价的现象。在存在外部性的情况下，边际社会成本要高于边际私人成本。网络非法有害内容的生产或传播存在明显的边际私人成本与边际社会成本不一致的情况。非法有害内容的生产者生产和上传非法有害内容会为自己带来较高的回报，而且在很多情况下会增加平台中介的用户流量，从而增加平台中介的收入，但非法有害内容的传播却可能会对知识产权权利人、社会特定群体乃至整个社会的公共利益造成损害。

由于数字内容平台传播非法有害内容给其他经济主体或者社会造成的巨大损害并不由其自身承担成本，因此其为了追求更高的收益往往会有较强的激励提供更多的非法有害内容以获得更多受众的注意。在图 15-1 中，数字内容平台生产或传播非法有害内容使全社会所承担的社会边际成本远高于数字内容平台的边际私人成本，数字内容生态环境中供应的非法有害内容数量（ $Q_{1}$ ）要大于市场最优数量（ $Q_{2}$ ）。因此，在不受监管的情况下，网络上会出现非法有害内容过度供给问题。

网络非法有害内容的传播不仅会污染网络生态环境，也会给社会带来严重的负外部性，主要体现为：一是影响社会稳定。如基于性别、宗教、民族、性取向等对特定群体发布歧视性或仇视言论，并在算法定向推送的助推下产生“信息茧房”效应，会进一步固化人们的偏见，引发极端言行，从而激发社会矛盾，影响社会稳定。二是形成不良的社会风

气。如功利主义、金钱至上、偏执的女权主义等有害内容的大肆传播，以及网络谣言、虚假新闻、机器生产的明星流量等，都会对社会价值观产生负面影响。三是影响个人身心健康。霸凌、辱骂、骚扰、谣言、自虐自残等信息内容以及一些让青少年沉迷的网络游戏会严重影响受众的身心健康，特别是儿童色情制品、性虐待和带有性暗示等内容的有害内容会给儿童的身心健康带来严重伤害。四是催生非法灰色产业。一些不法分子通常会在境内外采用一些技术手段以较低的成本搭建网络赌博平台或色情表演平台，进行淫秽色情表演、赌博、诈骗等，以牟取非法利益。五是威胁国家公共安全与政治安全。例如，涉及极端主义、恐怖主义的内容的传播会严重威胁公共安全，境外势力利用网络散播颠覆政治体制的言论会严重威胁国家政治安全。

# 15.4.4 网络非法有害内容监管的核心：强化数字内容平台主动治理责任

数字内容平台是网络信息内容传播扩散的中介，其在很大程度上能够掌控数字经济社会信息内容的传播。目前，数字内容平台日益成为影响人们沟通交流、文化传播、思想观念和伦理道德的重要力量。相对于政府监管，数字内容平台在治理非法有害内容方面具有明显的信息优势和技术优势，但由于平台以私人经济利益最大化为目标，并没有充分实施能实现社会公共利益最大化的有关非法有害内容治理的激励机制，导致消极治理往往成为一种常态，因此，强化数字内容平台主动治理责任成为实现网络信息内容有效监管的重要基础，是网络信息内容事前监管体制构建的核心。为此，需要重构以内容平台责任为核心的网络信息内容监管体制，通过平台责任配置来改变非法有害内容生产和传播主体的行为激励，构建清朗的网络空间和健康的平台生态，促进数字内容创新并维护社会公共价值。

欧盟2022年颁布的《数字服务法案》是全球第一部对数字内容平台传播的信息内容进行监管的法律，它为全球各国进行网络信息内容监管提供了重要的范本。该法案主要采取了分类监管政策并要求平台积极履行注意义务。

第一，针对不同类型平台提出了不同的义务要求。《数字服务法案》将在线信息中介服务提供商分为四类：（1）提供中介服务，如互联网接入服务、域名注册服务等的网络基础设施供应商；（2）托管服务者，如云处理服务、存储服务和虚拟主机服务等的提供者；（3）在线平台，如线上市场、应用商店、社交媒体平台；（4）超大型平台，即在传播非法内容和社会危害方面具有特殊风险的在线平台。《数字服务法案》对不同类型的中介服务提供者的义务提出了差别化要求（见表15-2）。

第二，激励数字内容平台积极履行注意义务。设计网络信息内容监管政策时，需要建立激励性监管体制，激励数字内容平台主动履行注意义务。一是数字内容平台主动过滤并删除非法有害内容。根据用户或受害方举报以及平台的识别手段，及时删除非法有害内容，是平台履行注意义务的主要机制。平台应建立非法有害内容风险管理机制，对非法有害内容进行风险评估与识别，对严重违法的非法有害内容应该及时屏蔽并立即删除，防止其上传或扩散传播，以积极履行过滤义务。二是数字内容平台依法履行“通知一行动”程序。数字内容平台应制定简便、透明的非法内容通知规则，并应及时告知内容提供商，并保证内容提供商有机会提出异议，以避免意外删除合法内容。

# 本章小结

网络与数据安全监管需求主要包括：一是维护国家安全与社会稳定；二是促进数字经济高质量发展。中国网络与数据安全监管政策有以下特点：确立了以国家总体安全观为统领的监管理念；构建了网络与数据安全监管的基础性法律体系；不断健全网络与数据安全监管体制；重视网络与数据安全技术保障能力建设。

个人隐私数据保护是促进数字经济发展的重要治理制度。由于个人隐私数据是个人重要的人格权，而通过市场机制无法提供有效的个人隐私数据保护，因此需要政府介入。中国个人隐私数据保护监管政策围绕完善个人隐私数据保护立法及加强隐私保护执法两方面进行。

数字经济反垄断政策的基本导向为：坚守竞争政策的基础性地位、坚持消费者福利标准并扩展对非价格因素的保护、将促进创新作为反垄断优先目标、重点关注数据和算法等方面的新型滥用行为。数字经济反垄断政策的重点为算法合谋、算法个人化定价、“二选一”行为、拒绝接入数据、数字平台并购。

强化网络信息内容监管、营造清朗的网络空间成为数字经济监管的重要内容。网络非法有害内容本质上属于负外部性问题。强化数字内容平台的治理责任是网络信息内容监管的核心，重点是建立分类配置的平台主体责任体系，并重点激励平台积极履行注意义务。
